# NutriMX v2.0 🧬

App de nutrición (PWA offline) con más de 100 alimentos con evidencia científica.

## Archivos

- `index.html` — App principal
- `app.js` — Lógica de la aplicación
- `ciencia.js` — Base de datos de alimentos con evidencia
- `sw.js` + `manifest.json` — PWA / funcionamiento offline
- `netlify.toml` — Configuración de deploy

## Deploy

Cada `git push` a `main` despliega automáticamente en Netlify.
